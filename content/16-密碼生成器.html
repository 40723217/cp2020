16-密碼生成器

首先我們來看看參考解答(雖然我的解答都是看解答然後去找怎麼做的):

#Python練習-16 PRACTICE PYTHON
#密碼生成器
#載入字詞生成系統
import string
#載入隨機系統
import random
#打印需要的密碼數量
print('How many characters in your password?')
#設計密碼計算方式
def password(size = 8, chars=string.ascii_letters + string.digits + string.punctuation):
    return ''.join(random.choice(chars) for _ in range(size))
print('請輸入數字:',end= '')
#輸入並打印密碼
print('這是你的密碼:'+password(int(input())))

首先我們來看看第一行

import string
這個意旨生成一些字元，參考PyMOTW的String項目可以知道，String可以這樣生成字元

ascii_letters='abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ'
ascii_lowercase='abcdefghijklmnopqrstuvwxyz'
ascii_uppercase='ABCDEFGHIJKLMNOPQRSTUVWXYZ'
digits='0123456789'
hexdigits='0123456789abcdefABCDEF'
letters='abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ'
lowercase='abcdefghijklmnopqrstuvwxyz'
octdigits='01234567'
printable='0123456789abcdefghijklmnopqrstuvwxyzABCDEFGHIJKLMNOPQRSTUVWXYZ!"#$%&\'()*+,-./:;<=>?@[\\]^_`{|}~ \t\n\r\x0b\x0c'
punctuation='!"#$%&\'()*+,-./:;<=>?@[\\]^_`{|}~'
uppercase='ABCDEFGHIJKLMNOPQRSTUVWXYZ'
whitespace='\t\n\x0b\x0c\r '
因此我們會需要它
接著是隨機的Random，參考python.org得知，Random有不少可以使用的指令
而參考答案中使用了random.choice，其原因是它可以更改一個字串的各個字元
random.choice（seq）
從非空序列seq返回一個隨機元素。如果seq為空，則引發IndexError。
接著就來搭配使用，先將這兩個系統import進來
並且也能讓使用者決定自己要用多少個密碼
接著開始定義密碼這個部分，使用def定義密碼password，後面開始設計密碼
這串密碼的代碼是string.ascii_letters·string.digits、string.punctuation所組成
所以所有符號都有包含在內

隨後讓使用者將輸入的數字變成字串數量，平均分配給每個字元
使用size變量然後讓random打散後，打印回給使用者
